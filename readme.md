<h1>Video Spliter</h1>  
<h3>Introduction</h3>  
This is a template of python video spliter project.  
Written by:  
<h4>Input/Output:</h4>
Input: video file need to split.
Output: image file (frame) of input video.
<h3>Installing</h3>  
Requirements:  
    python >= 3.5  
    opencv >= 3.3  

Install  
    pip3 install -r requirements.txt  

Run  
    python3 main.py
